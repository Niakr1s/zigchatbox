# zigchatbox

## Description
This application is made to help me learning zig just for fun.

## Usage example:
```
$ nc -U /tmp/zigchatbot.sock
Anon3: joined
Hello!
Anon2: Hello you!
/nickname User
> Anon3 is known as User now
> Anon2 is known as AnotherUser now
AnotherUser: I changed nickname too!
Cool
/who
> User
> AnotherUser
/whoami
> You are User
^C
```
